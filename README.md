# Message-box
A simple JavaScript project that demonstrates how to display message boxes using
both built-in browser dialogs and a custom message box.

## Features
- Alert message box
- Confirm dialog
- Prompt dialog
- Custom modal message box
- Easy to understand and beginner-friendly

## Technologies Used
- HTML
- CSS
- JavaScript



## Built-in Message Boxes
JavaScript provides three built-in message boxes:
- `alert()` – Displays a simple message
- `confirm()` – Asks the user to confirm an action
- `prompt()` – Takes user input

## Custom Message Box
A custom message box is created using:
- HTML for structure
- CSS for styling
- JavaScript for opening and closing the box

This provides better design control compared to built-in dialogs.

## How to Run
1. Download or clone the project
2. Open the HTML file in any web browser
3. Click the buttons to view different message boxes

## Example Use Cases
- Showing warnings or notifications
- Confirming user actions
- Collecting simple user input
- Displaying styled popups
